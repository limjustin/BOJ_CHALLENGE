### Commit

1. Connect account in repl.it
- ```git config --list```
- ```git config user.email "E-mail Address" ```

2. Force pull
- ```git fetch```
- ```git reset --hard origin/master```

3. Compile
- ```clang++-7 -pthread -o main (File root)```
- ```./main```

4. Upload
- Same as github upload
